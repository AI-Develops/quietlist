# QuietList

QuietList is a minimalist task list web app designed to help you focus without
noise, clutter, or unnecessary features.

The goal is simple: write things down, check them off, and move on.

## Features

- Distraction-free task list
- Keyboard-friendly task entry
- Tasks persist using localStorage
- Clean, responsive layout for mobile and desktop
- No frameworks, no build tools, no dependencies
